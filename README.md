## About
Life Hack or Urban Myth is a flash card game to help users test their common sense and learn useful and safe real world shortcuts.

## Features

Welcome screen with start button

Flashcard quiz with 6 questions and question counter to track progress

Instant feedback to each answer

Score screen with total score of quiz, feedback and review button

Review screen to see all correct answers

## Design

Navy blue and a bright orange for a sharp and bold look 

Used Luckiest Guy font for fun and energetc feel

Added an image to the welcome screen to make it visually appealing

Layout is kept simple to focus on the questions

## How to run the app

Clone the repository from GitHub

Open the project in Android Studio

Run the project on the emulator

## GitHub and GitHub Actions

A GitHub respository was created to store and track the project

Committed and pushed code to save progress

This ensures the app works not just on my computer
